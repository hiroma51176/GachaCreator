# ガチャクリエイター
ガチャクリエイターは自分だけのオリジナルガチャを作成したり、誰かが作成したガチャを無料で引くことができます。<br>
[ホームページ](https://infinite-reef-31165.herokuapp.com/)

## 機能一覧
- ガチャを引く
- ガチャのシミュレーション
- ユーザー登録、ログイン、ログアウト
- ガチャの作成、編集、削除
- プライズ（ガチャの景品）の作成、編集、削除
- ガチャを引いた結果の履歴確認

## 技術一覧
- 使用言語： HTML、CSS、JavaScript、PHP
- フレームワーク、ライブラリ： Laravel、jQuery
- ツール： AWS、Amazon S3
- デプロイ： Heroku
